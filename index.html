January 2019 - May 2019

A Machine Learning and deep learning approach for predicting Coronary Heart Disease (CHD) using Risk Factors

Guide: Dr. N. Kishorjit Singh, IIIT Manipur, India

Used various risk factors which are responsible for heart diseases such as age, sex, height, weight, heart rate, diabetic level, cholestrol level, etc used in predicting the CHD. The dataset for the project is taken from the UCI Machine Learning Repository https://archive.ics.uci.edu/ml/machine-learning-databases/arrhythmia/ . The System is trained on the dataset by applying classifiers such as Logistic regression, SVM, random forest, XGBoost etc. to predict other types of CHD (such as Tachycardia, Bradycardia and Coronary artery diseases using ECG signals) to make it more efficient. Later, the model was further improved by implementing deep learning architectures such as CNN, LSTM, Bi-LSTM and GRU.

September 2019 - December 2019

Finding out the needy one from Tweets : An analysis using #kerelafloods

Guide: Dr Navanath Saharia, IIIT Manipur, India

Worked on Identifying relevent words from Tweets to help People during disasters (flood). Implemented Topic Modelling to analyse the use of the system & Used K-Means, LDA. Implemented this project in order to help NGOs, Government agencies to help People in kerala during this destructive flood situation. Later, the system was further improved by implementing for indigenous people as now this tool extracts tweets based on various regional languages and it translates local languages(tweets) into English to be used by Government officials. Further a web tool is designed for proper visualisation of hashtags and information gathered from tweets during disaster.

Summer 2018

Guide: Dr. Anil Kumar Singh, IIT Varanasi (BHU), India

The aim of the project was to use Machine Learning algorithms for finding the solutions the problem. To make a Parallel Corpus from a Comparable Corpus obtained from wikipedia dumps for HungarianEnglish Language. Used Support Vector Machine, Logistic Regression and Random Forest Classifier on the training and testing data to achieve an accurate machine learning model.

Sentiment Analysis for racist/provocative words identification from Tweets
